# survival-typescript
learning for https://typescriptbook.jp/

## set up

### install node.js
- `brew install node@16`
- `echo 'export PATH="/usr/local/opt/node@16/bin:$PATH"' >> ~/.zshrc`
- `node -v`

### install typescript
- `npm install -g typescript`
- `tsc -v`
  
## run javascript
- `node increment.js`
